A custom plugin to create a new post type 'Events' and add additional fields.

== Description ==

AATI Events is a custom plugin that creates a new post type called 'Events'. This plugin adds additional fields for each event, such as:

* Link (more info or buy tickets)
* Start Date
* End Date

Additionally, the plugin provides an 'Add to Calendar' link using Google Calendar. It is translatable and compatible with WPML.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/aati-events` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Can I change the date format for the events? =

Yes, you can change the date format by adjusting the date conversion in the `aati_events_add_to_calendar_link()` function accordingly.
